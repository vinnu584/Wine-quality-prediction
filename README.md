# Wine Quality Prediction

A machine learning project that predicts wine quality using classification techniques.

## Features
- Trained ML model for wine quality prediction
- Flask web application for predictions
- Jupyter notebook with data analysis
- Docker support for deployment

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python app.py
```

Visit `http://localhost:5000` in your browser.

## Project Structure

- `Raw_Data/` - Training data
- `model.pkl` - Trained ML model
- `app.py` - Flask application
- `Wine_Quality_Prediction_Notebook.ipynb` - Analysis & training notebook
- `requirements.txt` - Python dependencies
- `Dockerfile` - Docker configuration

## Technologies Used

- Python 3.x
- scikit-learn
- Flask
- Pandas
- NumPy
